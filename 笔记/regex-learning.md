# 正则表达式
    以“*”开头的字符串：*?*

    表示一位数字：\d

    可能有一个负号，后面跟着一位或者多位数字：-?\\d+

    |表示或，(-|\\+)?表示字符串的起始位为一个-或者+

    任意字符：.

    包含a、b和c的任何字符：[abc]，a|b|c

    除了a、b和c以外的任何字符（否定）：[^abc]

    从a到z，A-Z的任何字符（范围）：[a-zA-Z]

    任意a，b，c，h，i和j字符：[abc[hij]]，a|b|c|h|i|j（合集）

    [a-z&&[hij]]：任意h，i或j（交集）

    \s：空白符

    \S：非空白符

    \d：数字[0-9]

    \D：非数字[^0-9]

    \w：词字符[a-zA-Z0-9]

    \W：非词字符[^\w]
